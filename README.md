Пошаговая инструкция по запуску проекта:

1. Скачайте и установите образ на виртуальную машину по ссылке: https://drive.google.com/file/d/1NgDEkE0HLVZqoJv8DdhWLCj2qPvdnbYA/view.
2. Скопируйте весь репозиторий.
3. Для генерации мира используйте файл rand_world.py, который можно запустить через консоль командой python3 rand_world.py. Это действие не обязательно, но может быть полезным.
4. Откройте симулятор Gazebo.
5. Запустите файл start.py для начала программы. Программа автоматически сгенерирует новый мир. Если мир окажется неподходящим, перезапустите процесс, запустив файл rand_world.py вручную и перегенерировав мир до тех пор, пока он не будет соответствовать требованиям.
6. Откройте веб-интерфейс через браузер по URL http://localhost:5000/ или http://127.0.0.1:8000/.
7. Нажмите кнопку «Старт» для запуска дрона. Если дрон долго не взлетает (около 10–15 минут), попробуйте запустить файл flight.py или отключите файл start.py, запустите файл app.py из папки app, обновите страницу и снова нажмите кнопку старта в веб-интерфейсе. Один из этих способов должен сработать.
8. После завершения миссии, когда дрон приземлится, запустите файл scan.py с помощью команды python3 scan.py.
