Пошаговая инструкция по запуску проекта:

Необходим установленный образ на виртуальную машину: https://drive.google.com/file/d/1NgDEkE0HLVZqoJv8DdhWLCj2qPvdnbYA/view

Скопировать весь репозиторий

Для генерации мира запустите через консоль файл rand_world.py для генерации мира, это можно сделать через команду python3 rand_world.py (может пригодится, не обезетельно)

Открыть симулятор gazebo

Запустить файл start.py для запуска программы (она сама сгенерит новый мир, если он сгенерирует мир неправельно то следует запустить файл rand_world.py вручную, и перегенирировать мир пока он не начнёт подходить для требований)

Открыть веб-интерфейс, можно сделать в браузере по URL http://localhost:5000/ или http://127.0.0.1:8000/

нажать старт для запуска дрона(если дрон прям долго(минут 10-15) не взлетает, то запустить файл flight.py)

после выполнения миссии(когда дрон сядет) надо запустить файл scan.py , это можно сделать через команду python3 scan.py
